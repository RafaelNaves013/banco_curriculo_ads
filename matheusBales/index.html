Aprendendo git e Github

Tópicos Especiais - ADS
  • $ git init (Inicia um repositório)
  • $ git add . (Adiciona as alterações para serem comitadas)
  • $ git status (Mostra status atual do repositório)
  • $ git config --local "variavel" (Altera as configurações para aquele repositório)
  • $ git config --global "variavel" (Altera as configurações para todos os repositórios)
  • $ git commit -m "Mensagem" (Realiza o commit para o repositório e adiciona uma mensagem)
  • $ git log (Exibe todas alterações feitas no repositório)
  • $ git restore --staged . (Remove todas as alterações adiconadas pelo git add .)
  • $ git reset --hard (Desfaz todas as alterações e volta para o ultimo commit)
  • $ git diff "Podendo filtrar por arquivo"(Mostra a diferença entre o ultimo commit e o que está para ser comitado)
  • $ git branch "nome"(Cria uma nova branch para realizar as alterações)
  • $ git switch "nome"(Altera para a branch desejada)
  • $ git merge "nome"(Traz as alterações da branch apontada para a branch atual)
